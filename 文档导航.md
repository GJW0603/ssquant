## 🎓 分类指南

### 安装与更新

> ⚠️ v0.4.3 起**不再支持 `pip install ssquant`**，必须通过 Git 或源码安装。

- 安装：`git clone` + `pip install -e .`
- 更新：`git pull origin main` + `pip install -e .`
- 详见 [README.md](README.md) 安装章节

### 数据相关

- API自动获取数据（松鼠俱乐部会员）
- 本地数据导入（见 `A_工具_导入数据库DB示例.py`）
- data_server 实时推送（WebSocket K线 + 订单流）
- 复权算法（前复权/后复权，本地计算）
- 三种数据请求方式（日期范围/精确时间/最近N根）

### 运行模式

- **回测配置**
  - 资金约束、拒单可见
  - 多品种多周期
  - 参数优化
  
- **SIMNOW模拟**
  - 账户配置
  - 服务器选择
  - 断线自动重连
  
- **实盘部署**
  - 实盘配置
  - 自动移仓
  - Tick抗压机制
  - 风险管理

### 高级功能

- **自动移仓** - 主力换月自动平旧开新
- **本地复权** - 前复权/后复权
- **实盘抗压** - 有界队列、压力等级感知
- **智能算法交易** - 限价单排队、超时撤单、追价重发
- **data_server** - 订单流数据、实时K线推送

## 🎯 快速查找

### 我是新手，从哪里开始？

1. 阅读 [README.md](README.md) 了解框架
2. 阅读 [用户手册.md](用户手册.md) "快速开始" 章节
3. 运行 `examples/B_双均线策略.py`
4. 参考 [API参考手册.md](API参考手册.md) 学习API

### 我要进行回测

1. 配置 `trading_config.py` 中的数据API账号
2. 参考 `examples/B_双均线策略.py`
3. 查看 [用户手册.md](用户手册.md) 回测模式章节

### 我要用SIMNOW

1. 在 simnow.com.cn 申请账号
2. 编辑 `trading_config.py` 配置账户
3. 修改示例策略的 `RUN_MODE = RunMode.SIMNOW`

### 我要部署实盘

1. ⚠️ 先阅读实盘风险警示
2. 先在SIMNOW充分测试（至少1周）
3. 查看 [用户手册.md](用户手册.md) 实盘部署章节

### 我要使用自动移仓

1. 查看 [用户手册.md](用户手册.md) 自动移仓章节
2. 参考 `examples/B_自动换月示例.py`
3. 在 `get_config()` 中设置 `auto_roll_enabled=True`

### 我要使用 data_server

1. 查看 [用户手册.md](用户手册.md) data_server模式章节
2. 参考 `examples/D_订单流与深度数据_data_server模式.py`
3. 在 `get_config()` 中设置 `kline_source='data_server'`
